# HWINFO Sensor Panel (Android)

Android application for real-time monitoring of PC sensors via HWiNFO64.

## Key Features:
- **Freeform Layout:** Drag & Drop tiles anywhere on the screen.
- **Tile Resizing:** Adjust width and height (1x1 to 4x4 grid cells).
- **Shapes:** Choose between Square, Circle, and Triangle tiles.
- **Deep Customization:** Custom colors for backgrounds, titles, and values. Individual font scaling.
- **Custom Background:** Support for gallery images with a blur (Glass Effect).
- **Clean UI:** All settings are hidden in a side-swipe drawer.

## Setup:
1. Run HWiNFO64 on your PC.
2. Enable Remote Sensor Monitor (web server).
3. Enter your PC's IP address in the Android app.

# HWINFO Sensor Panel (Android) - RU

Приложение для мониторинга показателей системы (CPU, GPU, RAM и др.) через HWiNFO64 в реальном времени.

## Основные возможности:
- **Свободный дизайн:** Плитки можно перетаскивать и размещать в любом месте экрана (Long Press для захвата).
- **Масштабирование:** Каждой плитке можно задать размер (ширина/высота в клетках).
- **Разнообразие форм:** Поддержка Квадратных, Круглых и Треугольных плиток.
- **Полная персонализация:** Настройка цвета фона, текста, размера шрифта для каждого датчика отдельно.
- **Кастомный фон:** Поддержка фоновых изображений из галереи.
- **Боковая панель:** Все настройки спрятаны в меню (свайп от левого края).

## Инструкция:
1. Запустите на ПК HWiNFO64.
2. Используйте Remote Sensor Monitor для раздачи данных по сети.
3. Введите IP-адрес ПК в приложении.
